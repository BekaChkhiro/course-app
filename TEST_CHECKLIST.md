## 🔍 ტესტირების ინსტრუქციები

### ვიდეო ტესტირება

1. გახსენით კურსი რომელშიც გაქვთ წვდომა
2. გადადით თავზე რომელსაც აქვს ვიდეო
3. ტესტირება:
   - **Swipe**: ვიდეოს არეალზე swipe left/right
   - **Double-tap**: ვიდეოს მარცხენა/მარჯვენა ნახევარზე ორჯერ შეხება
   - **Fullscreen**: Play ღილაკზე დაჭერა მობილურზე

### Review Modal ტესტირება

1. გაიარეთ კურსის ყველა თავი (ან გამოიყენეთ test account)
2. ბოლო თავის დასრულებისას უნდა გამოჩნდეს modal
3. შეავსეთ ვარსკვლავები და კომენტარი
4. გამოცდა უნდა გახდეს ხელმისაწვდომი

### Admin Grant ტესტირება

1. შედით admin account-ით
2. Admin panel → Students → აირჩიეთ სტუდენტი
3. "კურსის გააქტიურება" → აირჩიეთ კურსი → დაადასტურეთ
4. შეამოწმეთ:
   - სტუდენტს გაუჩნდა კურსი dashboard-ზე
   - Email მივიდა
   - Activity log-ში ჩაიწერა
